# 方太领导力学院 - 管理者的角色认知课程网站

## 1. Concept & Vision

一个专业、优雅的企业领导力培训课程网站，体现方太集团"仁爱"的企业文化精髓。网站以温暖的赤陶色为主调，传达教育与关怀的理念，让学习者感受到专业、系统、可信赖的课程体验。

## 2. Design Language

### 色彩方案
- 主色（Primary）: #C75C3A (温暖赤陶色)
- 次要色（Secondary）: #1A365D (深邃藏青)
- 强调色（Accent）: #D4A574 (暖金棕)
- 背景色（Background）: #FAFAF8 (温暖米白)
- 深色背景: #1E293B (深墨蓝)
- 文字色: #2D3748 (深灰)
- 浅文字: #718096 (中灰)

### 字体
- 标题：思源黑体 (Noto Sans SC), sans-serif
- 正文：思源黑体 (Noto Sans SC), sans-serif
- 装饰字体：Ma Shan Zheng (书法风格)

### 布局节奏
- 大量留白，强调内容层次
- 卡片式布局，圆角柔和（12px）
- 网格系统：响应式，桌面端最大宽度1200px

### 动效设计
- 入场动画：fadeInUp，时长0.6s，ease-out
- 悬停效果：scale 1.02，阴影加深
- 页面切换：平滑过渡

## 3. Layout & Structure

### 首页 (index.html)
1. **导航栏** - 固定顶部，透明背景，滚动后变白
2. **Hero区** - 全屏高度，左对齐文字，大标题+副标题+按钮
3. **统计数字** - 三个关键指标卡片
4. **课程卡片区** - 4个课程网格展示
5. **课程特色** - 4个特色图标卡片
6. **页脚** - 简洁链接

### 课程页 (course/index.html)
1. **课程标题区** - 大标题+简介
2. **课程目录** - 侧边导航
3. **课程内容** - 10个章节的详细内容
   - 一、不同发展阶段管理者的角色认知
   - 二、管理者的核心素养
   - 三、管理者的进阶路径
   - 四、管理者的三大角色定位：君、亲、师
   - 五、管理者的三大作风
   - 六、管理者的四大核心责任
   - 七、责任跃升的三个支撑
   - 八、关键理念与金句
   - 九、管理自己的五个维度
   - 十、核心工具与方法
   - 课程总结

## 4. Features & Interactions

### 导航
- 固定导航栏，滚动时背景变色
- 移动端汉堡菜单
- 平滑滚动到锚点

### 课程卡片
- 悬停时放大+阴影效果
- 点击进入课程详情

### 目录导航
- 固定侧边栏
- 当前章节高亮
- 点击滚动到对应章节

### 响应式设计
- 桌面：4列课程卡片，侧边目录
- 平板：2列卡片，目录收起
- 手机：单列，卡片堆叠

## 5. Component Inventory

### 导航栏 (Navbar)
- Logo + 品牌名
- 导航链接
- 响应式菜单按钮

### 按钮 (Button)
- 主按钮：赤陶色背景，白色文字
- 次按钮：透明背景，赤陶色边框
- 悬停状态：背景加深/边框加深

### 课程卡片 (CourseCard)
- 标签（壹/贰/叁/肆）
- 标题
- 描述
- 课时信息
- 难度标签
- 状态标签（已发布/即将上线）

### 特色卡片 (FeatureCard)
- 图标
- 标题
- 描述

### 章节块 (ChapterBlock)
- 章节编号（大字）
- 章节标题
- 内容（列表、表格、引用）

## 6. Technical Approach

- 纯HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，确保快速加载
- CSS变量管理主题色彩
- Flexbox + Grid布局
- Intersection Observer实现滚动动画
- 语义化HTML结构
